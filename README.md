O projeto tem como base a implementação de um jogo de caça-palavras na linguagem C usando uma árvore AVL para armazenar palavras encontradas e uma árvore digital (Trie) para validar possíveis palavras. O funcionamento baseia-se em ler os arquivos que guardam as palavras e o tabuleiro, e, logo após, verificar quais das palavras foram identificadas no jogo.

Sendo assim, ao executar o programa, o usuário possui as seguintes opções:

- Ler tabuleiro
- Ler palavras
- Buscar palavras no tabuleiro
- Listar palavras encontradas
- Sair
